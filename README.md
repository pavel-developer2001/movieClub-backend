# Backend version MovieClub

## Version 1.0.0

## Technology stack:

- NestJS
- Typescript
- TypeORM
- PostgreSQL
- Cloudinary

## Install

- 1 Install all packages:

```bash
npm i
# or
yarn install
```

- 2  Start application:

```bash
npm run dev
# or
yarn dev
```

The frontend version of this project [here](https://github.com/pavel-developer2001/movieClub-frontend)
